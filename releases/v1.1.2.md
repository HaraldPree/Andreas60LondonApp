## Was ist neu

`/api/version` zeigt jetzt zusätzlich:

```json
{
  ...
  "blobConfigured": true,
  "kvConfigured": true,
  "photoSharingReady": true
}
```

Hilft bei der Aktivierung von Vercel Blob + KV: wenn nur eines der beiden
konfiguriert ist, zeigt `hints[]` einen entsprechenden Tipp. So weiß man
sofort welcher Service noch fehlt.
