## Was ist neu

User-Wunsch: Gruppen-Konsens herstellen wenn jede:r seine eigene
Wunschliste markiert hat. Apple-Way-Lösung: **WhatsApp ist wo die
Gruppe eh kommuniziert** — App generiert nur fertig formatierten
Text, Voting läuft über Emoji-Reaktionen.

### Wie es aussieht

Im Wunschliste-Tab, rechts neben „Dein Stand · Harald":

```
┌─────────────────────────────────────────┐
│ Dein Stand · Harald   [📨 Poll teilen]  │
│ 💭 5  👁 4  ✓ 1                          │
└─────────────────────────────────────────┘
```

Button erscheint nur wenn **mindestens 1 💭-Wunsch** markiert ist
(sonst gäbe es nichts zu pollen).

### Was beim „Teilen"-Klick passiert

1. **iOS Safari / Chrome Mobile**: nativer Share-Sheet öffnet sich
   (`navigator.share`) → User wählt WhatsApp/iMessage/Mail aus
2. **Desktop / fehlender Web-Share-Support**: Fallback öffnet
   `wa.me/?text=…` in neuem Tab → User wählt WhatsApp-Empfänger
3. **Kopier-Button** als Backup falls Share nicht klappt

### 4 Template-Optionen

| Template | Wann sinnvoll |
|---|---|
| **Wer kommt mit?** | Tages- oder Ausflugs-Abstimmung (Default) |
| **Wohin essen?** | Wenn Liste Restaurants / Pubs enthält |
| **Top 3 voten** | Letzter Tag, Prioritäten setzen |
| **Eigener Text** | Frei formulierte Frage |

### Kein Backend nötig

Bewusste Apple-Way-Entscheidung: keine eigene Voting-Server-Logik. WhatsApp-
Emoji-Reaktionen sind das natürliche Voting-Interface der Gruppe — die App
muss nur einen ordentlich formatierten Text liefern.

Echter In-App-Group-Sync (Option C aus der Vorschlag-Iteration) bleibt für
v1.8.0 / Travel-Live Phase C — wenn dann ein Backend kommt, läuft das parallel.

## Was funktioniert NICHT (in dieser Version)

- **Voting-Auswertung manuell**: User muss WhatsApp selbst lesen + Konsens
  ablesen. Apple-Way: das passiert in echter Konversation, nicht in
  Zahlen-Spalten.
- **Keine Persistenz** des erstellten Polls in der App — wenn Sheet
  geschlossen, ist der Poll-Text weg (außer User hat in WhatsApp geteilt).
- **Nur eigene Wünsche pollen** — kann nicht „Andrea's Wünsche pollen"
  (kein Cross-Device-Read ohne Backend).
