## Was ist neu

### Neuer Ordner: `/todo/`
Git-tracked, oberhalb des Projekt-Roots. Enthält pro Verbesserungsvorschlag
eine eigene Markdown-Datei. Verhindert dass Verbesserungs-Ideen im
Chat-Verlauf verschwinden.

### Workflow
- Jeder Audit / Vorschlag / "machen wir später" → neues `.md` File in `todo/`
- Status-Tags: `proposed | planned | in-progress | done | rejected`
- Bei Umsetzung: Status auf `done` + Verweis auf die Release-Version
- Template-Format steht in `todo/README.md`

### Initial-Inhalt
- **`todo/v1.1.0-ux-audit.md`** — Vollständiges UX-Audit mit
  15 Punkten in 3 Prioritäts-Stufen (kritisch / wichtig / nice-to-have)
  + Cross-Platform-spezifische Punkte für iPhone / Android
- **`todo/firefox-mic-fallback.md`** — Mikrofon im KI-Chat funktioniert
  nicht in Firefox Android (Browser-Limit Web Speech API). Workaround
  via Keyboard-Mikrofon dokumentiert, 3 Code-Lösungs-Varianten skizziert.

### CLAUDE.md erweitert
Neuer Abschnitt im Release-Workflow erklärt die drei Doku-Ordner:

  (gitignored)
